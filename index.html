Estrutura básica do HTML

Voltar ao início

A estrutura basica de uma página html são as tags HTML, HEAD e BODY. A tag html delimita todo o código html. Dentro desta tag estarão as tags head e body. a tag head conterá outras tag de configuração da página htmk, como as tags meta, lang e title, entro outras.

A tag body contem toda a marcação da página que será mostrada no browser. Terão tags de formatação e estrutura como p, a href, ul, ol, e etc.

Falando sobre Tags

Voltar ao início

Tags são marcações num documento html. São as tags que darão formato e estrutura a página. As tags são escritas com o simbolo de <tag>, na maioria das tags há uma tag de fechamento que é ela mesma com uma barra, assim: </tag>.

Tags de textos

Voltar ao início

As tags de textos são usadas para formatar os textos. Também para dar formato ou função ao texto. A tag </p>, que tem a tag de fechamento </p>, é usada para delimitar um texto parágrafo. Coloca-se a tag <p> no inicio do texto e após o texto põe-se a tag de fechamento </p>, com isso o navegador vai pular uma linha para comoçar outro parágrafo ou para exibir outros elementos.

Também são tags de textos <h1>, <h2> até <h6>. Estas tag são tags de título, sendo o h1 o maior e de mais importancia, e vai diminuindo até chegar ao h6

Temos também as tags <strong>, <i> e <span> alem de outras

Tags de listas

Voltar ao início

As tags de listas servem para colocar textos em listas, como uma lista de compras, por exemplo. Ela começa com uma UL se for uma lista simples ou OL se for uma lista ordenada (numerada) e cada item da lista inicia com uma li. exemplo de listas:

    Lista Comum ou Não-ordenada
  • Primeiro item
  • segundo item
  • terceiro item

    Lista ordenada
  1. Primeiro item
  2. segundo item
  3. terceiro item

Links

Voltar ao início

Os links são ligações para outros sites, páginas ou localizações na própria página. O link começa com <a> e termina com </a>. No meio se coloca uma paravra ou mais que definen o link. A tag link tem atributos que são o href, que é o local do link, e o target, que diz onde será exibida a página.
